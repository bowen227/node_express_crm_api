CRM API Outline

List of routers needed

  • Contacts
    1. GET /contacts - returns all contacts
    2. POST /contacts - will create a new contact
    3. PUT /contacts - will update existing contact
    4. GET /contacts/:id - returns contact by id
    5. DELETE /contacts/:id - deletes a contact by id
  • Todo
    1. GET /todos - returns all to-do's
    2. GET /todos/:id - returns a to-do by id
    3. DELETE todos/:id - removes a to-do by id
    4. GET todos/contact/:contactId - returns all to-do's for a contact by id
    5. POST todos/contacts/:contactId - creates a new to-do for a contact
    6. PUT todos/contacts/:contactId - updates a to-do for a contact
